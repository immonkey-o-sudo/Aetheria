# Features
Aetheria is a feature-rich Quality-of-Life mod for Fakepixel Skyblock, which includes but is not limited to:

- **Storage Overlay** — manage your inventories with a custom overlay, with multiple themes and jump-to-active support
- **Item List & Recipe Viewer** — browse every SkyBlock item and view crafting recipes in-game
- **Custom Scoreboard** — fully configurable sidebar with custom lines, colors, ordering, and drag-to-reorder
- **Diana Tracking** — tracks burrows, playtime, mob rates, and loot with live HUD overlays
- **Dungeon Tools** — run timers, end-of-run stats, boss highlights, blood mob highlight, and a CS:GO-style chest opening animation
- **Chat Tools** — filters, compacting, timestamps, chat heads, copy, and animated messages
- **Mining Overlays** — powder tracker, pristine tracker, Fetchur overlay, commission highlights, and Pickobulus preview
- **Fishing** — trophy fish tracker and fishing timer
- **Farming** — mouse lock and BPS overlay
- **Waypoints** — ordered waypoints with auto-advance and a GUI manager
- **Profile Viewer** — `/pv [username]` to view SkyBlock stats in-game via SkyAtlas
- **Capes** — equip and display custom capes visible to other Aetheria users
- **And much more** — enchant parser, slot binds, item cooldowns, party commands, sign calculator, and dozens of other QoL improvements

A full list of features can be found [here](https://github.com/aetheria-org/Aetheria/blob/main/docs/FEATURES.md).

# 📋 IMPORTANT
When you join a server with Aetheria it sends some information to our servers. This information includes:
- Your Minecraft username
- The list of mods you have installed

This information is used for analytics and to track compatibility. We do not send any other information, such as passwords, files, chat messages or server IPs.

Some features require information to be sent. For example the capes feature sends your Minecraft username and the cape you have selected to a backend server. This allows other players to see your cape in the game. We only send the information that's necessary for a feature to work.

The information we collect is only accessible to the person who created the project. It is not shared with anyone.

The Aetheria (Formerly known as JustEnoughFakepixel) project is **Open source**, and you can review the source code to see exactly what information is sent and when.
